## توصيل الريلايات

### الريلاي الأول (GPIO 14)
```
ESP8266 GPIO 14 ────→ Relay 1 Signal Pin
                      ┌─────────────────┐
```

## حساس الشبكة العامة (GPIO 4)

```
                   Grid Power Detection Circuit
                   ============================

AC Grid Power ────→ ┌─────────────────┐
                    │   Transformer   │ (240V → 5V)
                    │    Rectifier    │
                    │    Optocoupler  │
                    └─────────────────┘
                            │
                         5V Signal ────→ ESP8266 GPIO 4
                            │
                           GND ────────→ ESP8266 GND

تفاصيل الدائرة:
- محول جهد 240V AC → 5V AC
- مقوم للتحويل إلى DC
- مقسم جهد للحصول على 3.3V
- اختياري: Optocoupler للعزل الكهربائي
```

## مقسم الجهد للبطارية (A0)

```
                    Battery Voltage Divider
                    =======================

Battery +ve (12V-48V) ────→ R1 (10kΩ) ────┬────→ ESP8266 A0
                                           │
                                          R2 (3.3kΩ)
                                           │
Battery -ve ──────────────────────────────┴────→ ESP8266 GND

حسابات المقاومات:
- لبطارية 12V: R1=10kΩ, R2=3.3kΩ (نسبة 3:1)
- لبطارية 24V: R1=22kΩ, R2=3.3kΩ (نسبة 7:1)  
- لبطارية 48V: R1=47kΩ, R2=3.3kΩ (نسبة 14:1)

جهد A0 = (Battery Voltage × R2) / (R1 + R2)
```

## مواصفات المكونات

### ESP8266 NodeMCU
- **الجهد**: 3.3V (مع منظم داخلي 5V)
- **التيار**: 80mA (عادي), 170mA (أقصى)
- **الذاكرة**: 4MB Flash
- **WiFi**: 802.11 b/g/n

### الريلايات
- **النوع**: SPDT (Single Pole Double Throw)
- **الجهد**: 5V Coil
- **التيار**: 30A @ 250V AC
- **العزل**: 4000V AC
- **المقاومة**: <100mΩ

### مقسم الجهد
- **R1**: 10kΩ (1/4W, 1% Precision)
- **R2**: 3.3kΩ (1/4W, 1% Precision)
- **الدقة**: ±0.1V
- **النطاق**: 0-32V (قابل للتعديل)

### حساس الشبكة
- **المحول**: 220V→5V, 1VA
- **المقوم**: 1N4007 × 4 (Bridge Rectifier)
- **التنعيم**: 100µF/16V Capacitor
- **الحماية**: Optocoupler PC817

## احتياطات السلامة

### ⚠️ تحذيرات هامة
1. **فصل الكهرباء**: افصل جميع مصادر الطاقة قبل التوصيل
2. **التأريض**: تأكد من التأريض الصحيح لجميع المعدات
3. **العزل**: استخدم عازلات مناسبة للجهود العالية
4. **الفصل**: استخدم قواطع دوائر مناسبة
5. **الاختبار**: اختبر النظام على أحمال صغيرة أولاً

### 🔧 أدوات مطلوبة
- مقياس فولت رقمي
- مقياس أمبير (Clamp Meter)
- لحام إلكتروني
- أسلاك مناسبة (14 AWG للأحمال العالية)
- صندوق توزيع مناسب

### 📋 خطوات التركيب
1. **تحضير المكونات**: فحص جميع المكونات
2. **التوصيل الأساسي**: توصيل ESP8266 ومقسم الجهد
3. **اختبار القراءات**: التأكد من قراءة الجهد صحيحة
4. **توصيل الريلايات**: بدون توصيل الأحمال
5. **اختبار التحكم**: التأكد من عمل الريلايات
6. **توصيل الأحمال**: توصيل الأحمال تدريجياً
7. **الاختبار النهائي**: اختبار النظام كاملاً
